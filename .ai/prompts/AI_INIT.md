# AI 작업 지침서 확인 테스트

- .ai 폴더를 먼저 읽는다, 가장 중요한 파일은 **PROMPT.md** 파일이다. 이 파일 내용은 반드시 지킨다!
- 간략한 작업을 통해 .ai 폴더를 숙지했는지 확인한다
- 작업 내용
  - .ai/tmp 폴더에 파일을 만들고, 현재 날짜를 기록하고, 10글자 아무 문장을 적는다
  - .ai/tmp 에 파일이 있는 경우 폴더 내용을 지우고 다시 작업한다

## 참고 사항
- **[중요]** : AI LLM 은 이 내용을 기억하지 않는다 !
  - 만약 테스트가 통과 되고 나면 LLM cli 사용자는 다음을 명령한다
  ```
  @.ai/prompts/AI_INIT.md 파일을 읽고 파일에 적힌 테스트를 먼저 진행해. AI_INIT.md 의 테스트 완료 이후 todolist 파일의 미완료 작업을 모두 진행해
  ```
